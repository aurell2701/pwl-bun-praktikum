# Praktikum Pemrograman Web Lanjut
Stack: Bun.js, Elysia, Tailwind CSS

# 1.Apa perbedaan Bun.js dan Node.js menurut pemahaman Anda?
Bun.js merupakan runtime yang lebih baru dan dirancang dengan fokus pada performa dan efisiensi. Bun.js dibuat menggunakan bahasa Zig sehingga mampu memberikan kecepatan yang lebih tinggi dalam menjalankan program, menginstal package, dan membangun aplikasi.

Sedangkan, Node.js adalah runtime yang sudah lama digunakan. Node.js dikenal stabil, banyak dokumentasi, serta didukung oleh banyak framework dan library.

# 2.Mengapa Git penting dalam pengembangan perangkat lunak?
Git digunakan untuk mengelola versi kode program, menjaga keamanan data, memudahkan kolaborasi, dan meningkatkan kualitas pengembangan software. Dengan Git, perubahan kode dapat dilacak, disimpan, dan dikembalikan jika terjadi kesalahan.
